 # QR Code Phishing URL Detection System 🚨


## Overview

This project combines the power of Artificial Intelligence, Machine Learning, and Cybersecurity to detect whether a QR code is benign or malicious. With the increasing use of QR codes in payments, authentication systems, advertisements, and websites, attackers often exploit them to redirect users to phishing pages, malware downloads, or fraudulent websites.

The system uses Machine Learning algorithms to analyze patterns and features extracted from QR code URLs and classify them as safe or malicious. By integrating AI/ML techniques with cybersecurity concepts, the project aims to provide an intelligent and automated approach for identifying potentially harmful QR codes before users interact with them.

This project demonstrates the practical application of AI in cybersecurity and highlights how machine learning can help improve digital safety and threat detection in real-world scenarios.🚀



## 🚀 Features

- 🔍 Detects whether a QR code is Benign or Malicious  
- 🤖 Uses Machine Learning for intelligent threat classification  
- 🔐 Integrates AI/ML concepts with Cybersecurity  
- 📊 Provides model evaluation metrics like Accuracy, Precision, Recall, and F1-Score  
- 📈 Generates Confusion Matrix for performance analysis  
- ⚡ Fast and automated QR code threat detection  
- 🧠 Helps identify phishing or malicious links hidden inside QR codes  
- 💻 Simple and easy-to-use Python implementation



## 🛠️ Tech Stack

- 🐍 Python  
- 🤖 Machine Learning  
- 🧠 Scikit-learn  
- 📊 Pandas & NumPy  
- 🔐 Cybersecurity Concepts
