#  Phishing URL Detection System

##  Overview
This project uses Machine Learning to detect whether a URL is phishing or safe.

##  Features
- ML-based phishing detection
- Dataset preprocessing
- Model training

## Tech Stack
- Python
- Scikit-learn
- Pandas

## How to Run
1. Clone the repository
2. Install requirements
3. Run train_model.py

##  Dataset
- Phishing URLs dataset
- Benign URLs dataset
